How it works
Upload any Chessable-style PGN where each line of an opening is a separate game.
Games sharing the same [White "..."] header are merged into one tree.
All comments {...}, variations (...), and headers are preserved.
Chapters titled Annotated/Model/Supplementary Games are left as-is.
Illegal moves and malformed tokens are safely converted to comments — the tool never crashes.
Output filename: <original name> merged.pgn
